- Integrate speech recognition tools like Whisper and text-to-speech engines.
- Analyze resumes and job descriptions using NLP techniques.
- Handle JSON and text file I/O for dynamic input/output handling.
- Collaborate on developing interactive interview simulations.

Required Skills:
- Python
- Natural Language Processing (NLP)
- Speech Recognition (e.g., Whisper, speech_recognition)
- AI Language Models (LLaMA, GPT)
- Flask or any backend framework
- JSON and file handling
- Version control with Git

Preferred:
- Knowledge of Docker and deployment on AWS
- Experience with React or similar frontend libraries

This is a project-based role, and performance during the internship may lead to future opportunities.
